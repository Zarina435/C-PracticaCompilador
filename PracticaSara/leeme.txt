# PracticaCompilacion
Práctica final C: compilador de un lenguaje usando Flex y Bison

Requisitos básicos realizados por:
Grupo 4: 
- David Bernal
- Sara Martín
- Xuban Barberena

Para compilar el programa bastará con escribir el comando make en la consola, y se ejecutará sobre las pruebas realizadas
El MakeFile se encuentra dentro de la carpeta "fuente".

Contenido del directorio fuente:
Programa principal y make: 
  - main.cp
  - Makefile
Ficheros flex y bison:
  - tokens.l
  - parser.y
 Funciones y objetos auxiliares:
 - Exp.hpp
 - Codigo.hpp
 - Codigo.cpp
